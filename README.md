## Descrição do Sistema

O **Vigilante** é um sistema web desenvolvido como evolução do projeto da Mostra STEAM.

O sistema simula um ambiente de monitoramento digital e segurança de usuários, com foco em autenticação, organização de páginas e experiência do usuário.

A aplicação foi construída utilizando Front-End e Back-End simples, com integração via API local.

---

## Objetivo do Projeto

O projeto tem como objetivo aplicar conhecimentos de desenvolvimento web criando um sistema funcional com:

- Cadastro de usuários  
- Login de usuários  
- Sessão de usuário  
- Navegação entre páginas  
- Integração Front-End + Back-End  
- Melhorias de UX/UI
